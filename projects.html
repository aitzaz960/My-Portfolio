Designing and Implementation of Sample Internet Banking System

In this project, I not only implemented a desktop app for sample internet banking system but also provided with all the documentation which is required for designing of any web/desktop app in the industry. The documentation for that includes the description, use cases, domain model diagrams, system sequence diagrams and sequence diagrams. For this project, I used Java Swing libraries for GUI, Intellij-idea as an IDE, MySQL for maintaining the databases and Java as a programming Language. In order to add the feature of security we will also be introducing the concepts of encryption, decryption and hashing to make the users data non compromised.

Designed Sample Internet Topology including DNS and Proxy server

In this Project I implemented a sample internet topology through socket programming which simulated how the communication between client and servers and various hosts take place. It also highlighted how the routing tables are shared, how the IP addresses are resolved and what role proxy servers play in these communications.

Designed CPU Scheduling Simulator

In this project, I designed CPU scheduling simulator by using the system calls provided by Linux in C/C++. This project simulated how a process goes through the 5 states (new, ready, running, blocked, exit) and how do these states affect the time a process takes to complete. In this project I implemented the scheduling algorithms of FCFS, SRTF, RR and SJF. This project used to read the information of processes (process name, arrival time, burst time) from a txt file and then simulate this process through the 5 states and at the end generate the results showing duration of the process for which it remained in different stats.

Designed Sample Linux Terminal

In this project, I designed a sample Linux terminal by using the system calls provided by Linux in C/C++. For this I used the concepts of multithreading, inter-process communication and synchronization techniques. This designed terminal was able to execute the basic Linux commands like change directory, list directories/files, clear screen, print working directory, top, ps, man and csh. In addition to that, this terminal was also able to perform input/output redirection and playing with environment variables.

Designed Database (including Front-end) for a school system

In this project, I designed a database for a case study of a school system. The case study required implementation of almost all of the functionalities which exist in real world school systems like enrollment of students, handling requests of section change, keeping information about the students for emergency cases etc. In this project, front-end was designed using HTML, PHP and CSS while in the backend, I used SQL/MySQL and Oracle as a DMBS. Through this project, I got to know about the problems like data redundancy which can come as a result of wrong design.

Designed Centipede Game

In this project, I designed an interactive one player game called centipede in which user has to try to stop its player from colliding with the weapons of the enemy and destroying his targets for as long as possible.
